# AGENTS.md

This project has a self-governing documentation system built for agents, by agents.

**Start at [`INTRODUCTION.md`](./INTRODUCTION.md).** It briefs you on the project and
the rules to follow, then points you to `HANDOFF.md` (current state + change log),
`DIRECTORY.md` (map of the codebase), and `SPEC.md` (the full product spec).

This file exists only because some coding-agent tools auto-discover `AGENTS.md` by
convention — it's a pointer, not the content. Don't add project detail here; add it to
the file it belongs in and update `HANDOFF.md` when you do.
