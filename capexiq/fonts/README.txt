Fonts per spec section 25.4:
  Inter/            - general UI text (alternative to IBM Plex Sans)
  IBM-Plex-Sans/    - general UI text (spec's primary pick)
  IBM-Plex-Mono/    - tabular financial numbers (payback, ROI%, NPV, IRR, etc.)

Source: Google Fonts (fonts.google.com)
License: SIL Open Font License 1.1 — free for commercial and personal use,
can be embedded/self-hosted, no royalties. Full license text ships inside
each family's original Google Fonts download; see https://scripts.sil.org/OFL

Weights included: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
Format: TTF (convert to WOFF2 with a tool like google-webfonts-helper or
fonttools for smaller web payloads when you actually build the site).
