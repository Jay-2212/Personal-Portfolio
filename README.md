# Personal Portfolio Website

This repository is for building and hosting a personal portfolio website.

## Goal
- Build an interactive personal website
- Host via GitHub Pages
- Serve through Cloudflare
- Use personal resume and photos as website content

## Project Assets
Upload your source materials to these folders:
- `resume/` → resume files (PDF/DOCX)
- `photos/` → profile photos and portfolio images

## Current Status
Initial project setup is complete with basic documentation and content folders.
