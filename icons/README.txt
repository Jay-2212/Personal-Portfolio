Icon set: Lucide (https://lucide.dev)
License: ISC (permissive, free for commercial use, no attribution required) — see LICENSE-lucide.txt
Format: SVG, single-color line icons (currentColor stroke) — recolor freely via CSS/props.

Folders map to the spec:
  equipment-clinical/  - equipment category icons (MRI/CT/scan, cath lab/heart-pulse, dialysis/droplets,
                          ultrasound/waves, lab/microscope+flask+test-tube, OT/cross, hospital/building, bed size)
  financial-model/     - ROI/NPV/IRR dashboard icons (calculator, rupee/dollar, trending, percent,
                          landmark=loan, wallet, piggy-bank, scale=break-even, pie/bar/line/area charts, gauge=score)
  ui-status/           - tooltip (help-circle), warnings (alert-triangle/circle-alert), success (circle-check),
                          risk (circle-x), advanced-mode toggle (sliders-horizontal), timing (clock/hourglass/calendar)
  export/              - Excel (file-spreadsheet), Word (file-text), ZIP (package), download/folder
  navigation/           - step-wizard icons (list-checks, circle-check-big), stakeholders (users), break-even (target)

Full library (1994 icons) is available anytime via: npm install lucide-static (or lucide-react for React components)
— only install that as a project dependency when you start coding; these ~60 are just the ones this spec calls for.
